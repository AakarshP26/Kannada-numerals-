# KTSD: Topology-Aware Feature Descriptors for Interpretable Kannada Handwritten Numeral Recognition

Code for the KTSD paper (under revision for IEEE Access). KTSD is a compact,
interpretable-by-design 29-dimensional feature descriptor for Kannada handwritten
numerals, built from topological and structural properties of the digit shape:
loops, stroke endpoints, junctions, stroke-direction statistics, and global shape
measures. An RBF-SVM trained on these features provides competitive accuracy at a
fraction of the cost and opacity of deep models.

> Status: this repository is being updated as part of the paper revision.
> Experiment protocols, baselines, and results will be finalized here, and every
> number reported in the paper is generated by code in this repository.

## Layout

```
src/            KTSD feature extraction
experiments/    Training / evaluation protocols
analysis/       Feature analysis and figure generation
app/            Interactive Streamlit demo
scripts/        Dataset download utilities (data is not stored in the repo)
results/        Metrics and generated figures
```

## Data

Experiments use [Kannada-MNIST](https://github.com/vinayprabhu/Kannada_MNIST)
(Prabhu, 2019): the official 60,000-image training split, the official 10,000-image
test split, and the harder out-of-distribution Dig-MNIST set (10,240 images).
Nothing is bundled here; see `scripts/download_data.py`.
